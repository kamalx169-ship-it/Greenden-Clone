 🌿 Greenden – Smart Plant Store

Greenden is a modern and responsive plant e-commerce frontend website designed to promote sustainable and eco-friendly living.
It showcases indoor and outdoor plants with a clean UI built using Tailwind CSS.

📌 Features

🌱 Modern and clean UI design

📱 Fully responsive (Mobile + Desktop)

🍃 Hero section with call-to-action buttons

🛍️ Best Seller product showcase

🌿 Why Choose Us section

⭐ Customer Review section

📧 Newsletter subscription UI

🍀 Eco-friendly brand theme

🛠️ Built With

HTML5

Tailwind CSS (CDN)

Google Fonts (Poppins)
